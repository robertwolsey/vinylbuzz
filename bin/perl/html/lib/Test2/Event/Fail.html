NAME

Test2::Event::Fail - Event for a simple failed assertion

DESCRIPTION

This is an optimal representation of a failed assertion.

SYNOPSIS

    use Test2::API qw/context/;

    sub fail {
        my ($name) = @_;
        my $ctx = context();
        $ctx->fail($name);
        $ctx->release;
    }
